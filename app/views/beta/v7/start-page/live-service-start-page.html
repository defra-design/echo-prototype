Apply for an export health certificate

You can use this service to:

  • apply for an export health certificate
  • copy existing applications
  • apply for blocks of certificates
  • apply for multiple certificates in a single application
  • see the status of your application
Start now

Before you start

You'll need a Government Gateway Account - if you don't already have one, you can set one up. You'll also need a Defra ID.

If you are an official vet or inspector, there's a different service to certify an export health certificate.

You can apply online for most exports. However, not every export health certificate is available to apply for using this service. Check the certificate you need is available.

Other ways to apply

You can still apply by sending your application to APHA by email or by uploading it for DAERA in Northern Ireland.
